## Getting Started

### Prerequisites

- JDK 21
- Docker & Docker Compose (for local DB)

### Run Locally

```bash
# 1. Clone repository
git clone https://github.com/clroot/selah-api-server.git
cd selah-api-server

# 2. Start database (Docker)
docker-compose up -d

# 3. Run application
./gradlew bootRun

# 4. Access API
curl http://localhost:8080/api/v1/health
```

### Build

```bash
# Build
./gradlew build

# Build without tests
./gradlew build -x test

# Run tests
./gradlew test
```
